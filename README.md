# skill-tom-de-voz
Skill para UX Writers e Content Designers validarem o alinhamento do tom de voz de conteúdos do Figma com a linguagem real do público-alvo. Compara textos de telas/fluxos com comentários dos últimos 10 posts do Instagram da iFood Brasil, identificando gaps e sugerindo ajustes para aproximar o conteúdo da forma como os usuários se comunicam.
