A integração da Apify no Claude não é uma "extensão" tradicional de navegador, mas sim configurada através do Model Context Protocol (MCP), que conecta o Claude Desktop (aplicativo de desktop) a ferramentas externas.
 
Aqui está o passo a passo para encontrar e configurar a extensão Apify no Claude:

1. Pré-requisitos

Claude for Desktop: Baixe e instale a versão de desktop do Claude (não funciona no navegador web diretamente para integrações locais).

Abra o Claude Desktop.

Vá para Settings (Configurações) no menu do aplicativo.

Selecione a aba Extensions (Extensões).

Clique em "Browse extensions" (Navegar por extensões) para visualizar o diretório curado pela Anthropic.

Clique em "Install" na extensão desejada.
