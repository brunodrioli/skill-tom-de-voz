Para conseguir sua API Key na Apify, acesse o Console da Apify https://console.apify.com/, 
vá até a seção Settings (Configurações) e clique na aba Integrations (Integrações), 
onde você encontrará seu token de API para autenticação e uso nos seus projetos. 

Passos para obter sua API Key:

- Faça Login: Acesse sua conta na Plataforma Apify.

- Vá para Configurações: No painel principal, procure e clique em Settings (Configurações) no menu lateral.

- Encontre Integrações: Dentro das configurações, selecione a aba Integrations (Integrações).

- Copie o Token: Seu token de API estará visível lá.
